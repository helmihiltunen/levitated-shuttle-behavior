Python program used to control ACOPOS 6D simulation in Planar Motor Simulation Tool and Planar Motor Tool, developed for the Bachelor's thesis "Traffic Policies and Behaviour Ratios for 6-DoF Levitated Shuttles" at Tampere University (2026).
Requirements: Python 3.11, pmclib, Planar Motor Simulation Tool, Planar Motor Tool
Usage: Configure CONFIG and PUSHER_RATIO variables at the top of the file before running.
